## **알고리즘 풀이 방식 분류**

여러 알고리즘 문제를 풀기 위한 다양한 방식이 존재한다.
어떤 상황에서 **어떤 알고리즘**을 적용해야하는지 알아보자.

---

### 모듈러 연산 **`%`**

- 기본적으로, 어떤 숫자를 <span class="yellow2pen">다른 숫자로 나눈 나머지</span>를 구할 때   
- <span class="yellow2pen">데이터가 순환하여 반복하는 구조</span>인 경우(원형적 자료구조)

### 에라토스테네스의 체

- <span class="yellow2pen">여러 개의 수가 소수인지 판별</span>할 때

### 이분 탐색

- 주어진 <span class="yellow2pen">데이터와 구하려는 값이 서로 **반비례** 관계</span>일 때

### 누적합

- <span class="yellow2pen">수열의 특정 구간의 합</span>을 빠르계 구할 때

### 슬라이딩 윈도우

- <span class="yellow2pen">**고정적인 개수의 구간**의 합</span>을 빠르계 구할 때

### 투 포인터

- <span class="yellow2pen">**가변적인 개수의 구간**의 합</span>을 빠르계 구할 때

### 그리디(탐욕법)

- <span class="yellow2pen">부분적인 단계에서 최선의 선택</span>을 하며 최종 해를 구할 때  

### 브루트포스(완전탐색)

- 가능한 <span class="yellow2pen">모든 경우의 수를 구하여 도출</span>할 때   

### 백트래킹(가지치기)

- <span class="yellow2pen">조건에 만족하지 않는 경우를 걸러내고 탐색</span>할 때
- <span class="yellow2pen">완전탐색보다는 효율적인 탐색이 필요</span>할 때

---

## **가중치에 따른 풀이 방식 분류**

> 알고리즘 문제를 **그래프**에 적용할 때의 해결 방법을 나열한다.  

### BFS(너비 우선 탐색) & DFS(깊이 우선 탐색)

- <span class="yellow2pen">**가중치가 하나**로 고정되는 경우</span>

### 0-1 BFS(Deque 사용)

- <span class="yellow2pen">**가중치가 0 또는 1**만 있는 경우</span>

### Dijkstra(다익스트라)

- <span class="yellow2pen">**가중치가 여러개**이고, **0 또는 양수**로만 이뤄져 있는 경우</span>

### Bellman Ford(벨만 포드)

- <span class="yellow2pen">**가중치가 여러개**이고, **음수**도 포함되는 경우</span>

### Floyd Warshall(플로이드 워셜)

- <span class="yellow2pen">**가중치가 여러개**이나, **임의의 경로**만 찾는 경우</span>

### DP(동적 계획)

- **순환성이 없는 그래프**에서, <span class="yellow2pen">이전 값을 가져와 다음 값을 찾는</span> 경우
- **단방향성을 띠는 자료 형태**에서 사용
