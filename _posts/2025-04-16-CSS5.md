## **CSS-IN-JS 방식의 스타일링**

- 말 그대로, **<span class="yellowpen">자바스크립트</span>** 코드 안에 CSS를 작성하는 방식.
- 스타일을 **<span class="yellowpen">자바스크립트</span>** 코드 안에 정의하고 **컴포넌트**와 함께 번들링하는 접근 방식.

### **styled-components**

- 공식 홈페이지 존재 → 설치 및 사용법 안내   
    
    [styled-components](https://styled-components.com/)
    
- 설치하기   
    
    ```bash    
    npm install styled-components
    ```   
    
    <img width="319" height="153" alt="Image" src="https://github.com/user-attachments/assets/fbafac98-97c3-4b2d-b0fe-5f9f67f519b1" />   

- 가이드 라인에 따라, 작성 후 실행   
    
    <img width="1303" height="558" alt="Image" src="https://github.com/user-attachments/assets/fd7aeb1e-bdff-4321-9545-2dd82984dcaf" />     
       
    ```tsx
    import styled from "styled-components";
    
    export default function App() {
      const Title = styled.h1`
        font-size: 1.5em;
        text-align: center;
        color: #bf4f74;
      `;
    
      return (
        <>
          <Title>App Component</Title>
        </>
      );
    }
    ```
    
- **자세한 과정**    
    
    1. `styled-components` 패키지 불러오기  
    2. 식별자에 특정 태그를 지정하고 옆에 **백틱(`)**을 사용하여 css를 적용한다.    
       → **태그드 리터럴 템플릿**    
    3. **스타일 컴포넌트명**을 **<span class="orangepen">JSX</span>**에 작성하여 나타낸다.

### **emotion**

- 공식 홈페이지 존재 → 설치 및 사용법 안내
    
    [Emotion – Introduction](https://emotion.sh/docs/introduction)
    
- 설치하기
    
    ```bash
    npm i @emotion/css
    # npm i @emotion/react
    ```   
    
    <img width="552" height="238" alt="Image" src="https://github.com/user-attachments/assets/783f52eb-b1b9-4e59-8c65-abcd12d10432" />     

- 가이드 라인에 따라, 작성 후 실행   
    
    <img width="908" height="578" alt="Image" src="https://github.com/user-attachments/assets/8f48568f-b1b0-460b-aa06-557b6a708001" />    
    
    ```tsx  
    import { css } from "@emotion/css";
    
    export default function App() {
      return (
        <>
          <h1
            className={css`
              color: red;
            `}
          >
            App Component
          </h1>
        </>
      );
    }
    ```    
    
    기존 **styled-components**의 <span class="redpen">태그를 알아볼 수 없다는 **단점**</span>을 보완.
    
- **자세한 과정**   
  
    1. `@emotion/css` 패키지 불러오기     
    2. 태그의 클래스명을 지정할 때, `css`를 적고 **백틱(`)**을 사용하여 css를 적용한다.

### **Vanilla Extract**

- **styled-componenets, emotion** 방식과 달리, **<span class="yellow2pen">제로 런타임</span>**.
    - 런타임 전에, css 적용됨.    
- **<span class="bluepen">타입스크립트</span>** 문법 지원.   
- 공식 홈페이지 존재 → 설치 및 사용법 안내    
    
    [Vanilla-extract — Zero-runtime Stylesheets-in-TypeScript.](https://vanilla-extract.style/)
    
- 설치하기    
    
    ```bash
    npm install @vanilla-extract/css
    ```  
    
- 추가로, 번들러 통합을 위한 설치 [**<span class="purplepen">vite용</span>**]    
  [Vanilla-extract(Vite용)](https://vanilla-extract.style/documentation/integrations/vite/#vite)   
    
    ```bash  
    npm install --save-dev @vanilla-extract/vite-plugin
    ```  
    
- 설치 확인
    
    <img width="406" height="288" alt="Image" src="https://github.com/user-attachments/assets/29592047-5a24-4ffe-aee6-0ccc563e21cf" />

    
- 플러그인 설정 - `vite.config.js(ts)` [**<span class="purplepen">vite용</span>**]
    
    ```tsx
    import { vanillaExtractPlugin } from '@vanilla-extract/vite-plugin';
    
    export default {
      plugins: [vanillaExtractPlugin()]
    }   
    ```   
    
- 가이드 라인에 따라 작성 후, 실행.      
  → `*.css.ts`의 형식을 파일 생성 후, 작성.  
    
    <img width="815" height="563" alt="Image" src="https://github.com/user-attachments/assets/0b749e62-8efd-4208-963a-76e9582ab4df" />     

- **자세한 과정**    
  
    1. **css 폴더 하위**에 `*.css.ts`를 생성하여 다음 코드를 작성
        
        ```tsx
        import { style } from "@vanilla-extract/css";
        
        export const container = style({
          backgroundColor: "red",
          color: "white",
          padding: 10,
        });
        ```
        
        - `@vanilla-extract/css` 패키지 불러오고,   
        - 식별자에 `style()`안의 **객체 형식**으로 스타일링 후, 내보내기.     
    
    2. 스타일링한 **타입스크립트 파일**을 불러와서, 스타일 적용.   
        
        ```tsx
        import { container } from "./css/styles.css.ts";
        
        export default function App() {
          return (
            <>
              <h1 className={container}>Vanilla Extract</h1>
            </>
          );
        }
        ```
        
        - 클래스명에 추가. (데이터바인딩 활용)
