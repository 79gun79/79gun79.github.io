## <span class="greenback">데이터바인딩 문법</span>

💡 **\{\{데이터바인딩\}\}** : JS 데이터를 HTML에 넣는 문법

- 기존 자바스크립트 스타일
  ```javascript
  document.getElementById().innerHTML;
  ```
- Vue.js 스타일 : <script>에 해당 문법 추가
  ```javascript
  data() {
    return {
      // 데이터저장(object형식으로)
      변수1 : 데이터1,
      변수2 : 데이터2,
      변수3 : [데이터31,데이터32,데이터33]
      }
    }
  ```

![뷰미지10](../../../../assets/images/vuenote/vueimg10.png)

- HTML에서 데이터 사용 : \{\{데이터바인딩\}\}
  ```html
  \{\{변수명\}\}
  ```
  사용이유 : 하드코딩 어려운 점을 극복, 실시간 자동 렌더링 사용을 위함
- HTML 속성 데이터바인딩
  ```html
  <태그명 :속성명="변수명"></태그명>
  ```
  > 앞에 :(콜론)을 붙여준다.

## <span class="greenback">반복문</span>

Vue의 HTML 반복문 : v-for

```html
<태그 v-for="반복명 in 반복횟수" :key="반복명">반복명</태그>
<태그 v-for="반복명1, 반복명2 in 반복횟수" :key="반복명1">반복명1</태그>
// 일반적인 반복

<태그 v-for="반복명 in 배열명" :key="반복명">\{\{반복명\}\}</태그>
// 배열요소 반복
```

> key도 무조건 쓸것

- **:key**의 용도 ⇒ 반복문 돌린 요소를 구분하기 위함.

## <span class="greenback">이벤트 핸들러</span>

💡 버튼 클릭과 같은 이벤트 시, 자바스크립트 실행

- 기존 자바스크립 문법
  ```javascript
  <button onClick="js코드"></button>
  ```
- Vue 문법
  ```vue
  <button v-on:click="js코드;"></button>
  <button @click="js코드;"></button>
  ```

<br>

⇒ 변수 증가 이벤트 만들기

```vue
<button v-on:click="증가변수 += 1">버튼이름</button>
<p>증가수 : \{\{증가변수\}\}</p>
```

- 이벤트 종류  
  click : 클릭  
  mouseover : 마우스 갖다대기  
  input

### 간단한 JS 문법

- 기존 자바스크립트
  ```javascript
  <script>
  function 함수명() {...js코드...}
  </script>
  ```
- Vue 문법

  ```vue
  <script>
  export default {
  methods : {
  	함수명() {...js코드...}
    }
  }
  </script>
  ```

  <br>

  **<span class="redst">주의사항</span>** : 함수에서 변수 사용시 this를 붙여줘야함!

  ```vue
  <script>
  export default {
    methods: {
      함수명() {
        this.변수명 += 1;
      },
    },
  };
  </script>
  ```

## <span class="greenback">서버에서 데이터 가져오기</span>

💡 자바스크립트 import, export 문법 사용

- export (데이터 내보내기)

  ```jsx
  var 변수명1 = 데이터;

  export default 변수명1(배열명1);
  // 1의 사용법

  export { 변수명1, 변수명2 };
  // 2의 사용법
  ```

  <br>

- import (데이터 가져오기)

  ```jsx
  import 변수명1(배열명1) from '파일경로명';
  // 사용법 1 사용 시

  import {변수명1, 변수명2} from '파일경로명';
  // 사용법 2 사용 시
  ```

  <br>

- [ ]안의 객체 데이터 사용방법  
  <span class="orgback">데이터 형식</span> ⇒ 변수명 = [{title:~, name:~,…},{ },{ },{ },{ }]  
  <span class="orgback">데이터 사용</span> ⇒ 변수명[0].title 식으로 사용
