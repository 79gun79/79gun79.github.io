### **Tailwind CSS 특징**

**Utility-First CSS**: 스타일을 적용하기위한 속성과 값이 **클래스 단위**로 분리하여 나뉨.

- **재사용성**: 각 클래스가 독립적이고 재사용 가능하므로,    
  <span class="bluepen">필요할 때마다 적절한 클래스를 조합해서 사용 가능.</span>    
- **코드 양 감소**: HTML 파일에서 직접 클래스를 조합하여 스타일을 지정하므로     
  <span class="bluepen">별도의 스타일 시트를 유지할 필요가 없음.</span>   
- **빠른 개발**: 미리 정의된 유틸리티 클래스를 바로 사용할 수 있어,    
  <span class="bluepen">스타일을 빠르게 적용 가능.</span>    

**Tailwind CSS 클래스명**

1. 속성이 항상 앞에 제시되어 있음.
2. 속성에 대한 값은 하이픈(-)으로 엮어서 제시됨.
3. 사용자 원하는 값을 제시할 수 있는 **커스텀 클래스**도 가능.

## **Tailwind 기본 스타일**

### **Tailwind Preflight**

[tailwindcss, Preflight - Base styles](https://tailwindcss.com/docs/preflight)    

- Tailwind의 가장 근간이 되는 기본 스타일
- 어느 브라우저에서 열든 일관적으로 보이도록 하는 스타일

---

1. 모든 HTML 요소에 적용된 `margin`, `padding` 모두 제거
2. 모든 HTML 요소에 적용된 `box-sizing` 속성을 **`border-box`**로 설정
3. 모든 HTML 요소의 테두리 스타일이 재설정   
    
    ```css
    border: 0 solid;
    ```  
    
    이 재설정으로 `border`로만 클래스를 추가하면,   
    **현재 색상**을 사용하는 **1px**의 단색 테두리가 추가될 수 있음.   
    
4. `<h1>`, `<h2>`, `<h3>`와 같은 제목 요소에 적용된 기본 스타일 제거    
5. `<ul>`, `<ol>`와 같은 목록 요소에 적용된 기본 스타일 제거    
6. `<img>`, `<video>`와 같은 멀티미디어 요소의 `display` 속성을 **`block`**으로 설정  
7. `<img>`, `<video>`와 같은 멀티미디어 요소가     
   부모 요소를 삐져나갈 수 없도록 <span class="redpen">너비를 제한</span>   

<br>

---

### **Preflight 확장**

- 하지만, 제목 요소와 같은 경우에는 스타일 제거를 원하지 않을 수 있음.
- 이와 같은 경우에는 Preflight를 확장시킬 수 있음!
- 다음과 같이 tailwindcss에, 기본 스타일을 확장.
    
    ```css
    @layer base {
      h1 {
        font-size: var(--text-2xl);
      }
      h2 {
        font-size: var(--text-xl);
      }
      h3 {
        font-size: var(--text-lg);
      }
      a {
        color: var(--color-blue-600);
        text-decoration-line: underline;
      }
    }
    ```

### **컴포넌트 클래스 추가**

- **자주 사용하는 CSS**를 모아, 클래스를 추가할 수 있음.
    
    ```css
    @layer components {
      .새로운클래스명 {
      @apply ...tailwind의 유틸리티 모음...;
      }
    }
    
    @layer components {
      .새로운클래스명 {
        ...css 속성과 값 모음...;
      }
    }
    ```

### **유틸리티 추가**

- Tailwind에서 제공하는 유틸리티 외에도 **사용자 지정 유틸리티**를 추가할 수 있음.
    
    ```css
    @utility 새유틸리티명 {
      content-visibility: auto;
    }
    ```
    
    위와 같이 지정하면, HTML에서도 유틸리티 사용 가능  
    
    - 또한, `hover`, `focus` 와 같은 변형도 적용 가능.

### **tailwind-merge**

- tailwind를 쉽게 사용할 수 있도록 도와주는 패키지   
    
    [npm, tailwind-merge](https://www.npmjs.com/package/tailwind-merge)   
    
- <span class="yellow2pen">매개변수로 전달된 클래스명을 합쳐줌</span>       
- <span class="yellow2pen">합칠 때, 역할이 중복되는 클래스는 제거</span>     
    
    <img width="415" height="69" alt="Image" src="https://github.com/user-attachments/assets/da579419-c12c-43f7-9be7-fb91decfa44c" />   

    <span class="graypen">적용 전</span>  
    
    <img width="396" height="68" alt="Image" src="https://github.com/user-attachments/assets/ba5fe08f-58b1-4b5b-a1a5-5ddbfb75b914" />  
    
    <span class="graypen">적용 후(중복 제거)</span>    
    
- 사용을 위한 설치
    
    ```bash
    npm i tailwind-merge
    ```
    
- **사용**: 패키지를 불러오고, **<span class="purplepen">twMerge</span>()**에 `콤마(,)`로 구분하여 작성    
    
    ```tsx
    import { twMerge } from "tailwind-merge";
    
    export default function App() {
      return (
        <>
          <h1 className={twMerge("text-3xl", "text-9xl")}>
            Hello world!
          </h1>
        </>
      );
    }
    ```
