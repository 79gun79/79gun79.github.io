## **JSX 문법**  

### **<span class="yellowpen">JavsScript</span> + <span class="redpen">XML</span> (확장 문법)**   

- **<span class="orangepen">JSX</span>**는 **HTML**과 굉장히 유사   
- **<span class="bluepen">리액트 개발팀</span>**이 편하게 작성하도록 제작.   
- JSX 범위    
    
    <img width="724" height="292" alt="Image" src="https://github.com/user-attachments/assets/4ca33db7-2d75-4436-8903-80fb877a20cb" />

### **1. 반드시, <span class="redpen">하나의 루트 요소만 반환</span>해야 함!**   

- 루트 요소를 2개 이상으로 두면, <span class="redpen">문제가 발생</span>한다.     
    
    ```tsx   
    export default function App() {
      return (
        <div>하나</div>
        <div>둘</div>       // ✖
      );
    }
    
    export default function App() {
      return (
        <div>
          <div>하나</div>
          <div>둘</div>     // ✔
        </div>
      );
    }    
    ```    
    
- 그러나, 이러한 규칙 때문에 <span class="redpen">불필요한 태그를 생성</span>해야 한다.      
- 따라서, 이 문제를 해결하기 위한 대책으로 **<span class="greenpen">React.Fragment</span>**가 있다.    
    
    ```tsx  
    import React from "react";
    
    export default function App() {
      return (
        <React.Fragment>
          <h1>App Component</h1>
          <h1>App Component</h1>
        </React.Fragment>
      );
    }
    ```   
    
    > 하지만 이 방식은 **React**를 `import`해야하는 귀찮음이 존재한다.   
     
- 더 편한 방식인, **빈태그 `<>`**가 있다.
    
    ```tsx    
    export default function App() {
      return (
        <>
          <h1>App Component</h1>
          <h1>App Component</h1>
        </>
      );
    }
    ```   
     
---

### **2. 모든 태그는 <span class="yellow2pen">닫아야 함!</span>**

- 다음과 같이 태그는 닫아주도록 한다.  
    
    ```tsx  
    export default function App() {
      return (
        <>
          <h1>App Component</h1>
          <h1>App Component</h1>
          <br/>
        </>
      );
    }
    ```    
    
---

### **3. 모든 태그의 속성은 <span class="yellow2pen">카멜 케이스로 작성</span>해야 함!**   

- 기존의 **HTML**에서 작성하던 <u>속성명은 소문자로만 구성</u> → <span class="redpen">그대로 작성시, 오류</span>.   
    
    <img width="304" height="117" alt="Image" src="https://github.com/user-attachments/assets/d978608a-32c1-4a97-87e1-8789c6c49313" />   
    
- 따라서, **카멜 케이스**로 작성한다!   
    
    ```tsx   
    export default function App() {
      return (
        <>
          <h1>App Component</h1>
          <h1>App Component</h1>
          <br />
          <input type="text" readOnly />
        </>
      );
    }
    ```  
    
- 또한, 기존에 **HTML**에서 알고 있던 <span class="redpen">**예약된 속성명**을 쓸 수 없다</span>!      
  따라서, 사용 시에 수정할 수 있도록한다.       
    
  <img width="407" height="176" alt="Image" src="https://github.com/user-attachments/assets/feaf776c-b2da-4252-af0e-a3e9fac1c100" />   
    
  ```tsx 
    export default function App() {
      return (
        <>
          <h1>App Component</h1>
          <h1 className="tite">App Component</h1>
          <br />
          <label htmlFor="uname">userId: </label>
          <input id="uname" type="text" readOnly />
        </>
      );
    }
    ```   

---
    
### **4. 표현식은 <span class="bluepen">중괄호</span> `{}`로 출력!**

- **데이터 바인딩**을 중괄호 안에 넣어서 진행.    
    
    ```tsx    
    export default function App() {
      const name = "jaegeon";
      return (
        <>
          <h1>10 + 20 = {10 + 20}</h1>
          <h1 className="tite">{name}</h1>
          <br />
          <label htmlFor="uname">userId: </label>
          <input id="uname" type="text" readOnly />
        </>
      );
    }
    ```    

---    

### **5. 인라인스타일 적용 시, <span class="yellow2pen">객체로 표현</span>!**

- 기존의 **HTML**에서 <u>인라인스타일 적용 시, 문자열로 명시</u>했다. → <span class="redpen">그대로 작성시, 오류</span>.     
    
    <img width="632" height="173" alt="Image" src="https://github.com/user-attachments/assets/2746f51c-8e0b-42de-81ff-60404e0c385c" />   
    
- 따라서, 문자열이 아닌 **객체**<u>로 나타내도록 한다</u>.    
    
    ```tsx   
    export default function App() {
      const name = "jaegeon";
      const style = {
        color: "red",
        fontSize: "24px",
      };
      return (
        <>
          <h1 style={{ color: "blue", fontSize: "24px" }}>10 + 20 = {10 + 20}</h1>
          <h1 className="tite" style={style}>
            {name}
          </h1>
          <br />
        </>
      );
    }
    ```

### **6. 주석 작성법**

- 주석은 다음과 같이 나타낼 수 있다.   
    
    ```tsx   
    export default function App() {

    			{/* 주석 작성법 */}
          <h1>App Component</h1>
          <h1>App Component</h1>
        </>
      );
    }
    ```

## **JSX 문법이 없었다면**

- 기존의 **React**에서, 문서를 작성하려면 다음과 같이 작성했음.   
    
    ```tsx  
    import React from "react";
    
    export default function App() {
      return React.createElement(
        "div",
        { className: "title" },
        React.createElement("h1", null, "App Component")
      );
    }
    ```  
    
- 같은 내용을 **<span class="orangepen">JSX 문법</span>**을 사용 하면? → 훨씬 간단.   
    
    ```tsx   
    export default function App() {
      return (
        <div className="title">
          <h1>App Component</h1>
        </div>
      );
    }
    ```
